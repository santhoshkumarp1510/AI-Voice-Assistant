# 🎙️ AI Voice Assistant

An intelligent **AI-based Voice Assistant** built using Python that can understand voice commands, perform tasks, and provide interactive responses. This project integrates multiple advanced features like **speech recognition, face authentication, gesture control, and screen reading** to create a smart assistant system.

---

## 🚀 Features

* 🎤 **Voice Command Recognition**
* 🧠 **AI-Based Response System**
* 🔐 **Face Authentication for Security**
* ✋ **Gesture Control Integration**
* 🖥️ **Screen Reading Capability**
* ⚡ **Real-time Command Processing**

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries & Tools:**

  * Speech Recognition
  * OpenCV (Face Detection)
  * Text-to-Speech Engine
  * AI APIs (Groq / LLM integration)
* **Other Tools:** Git, GitHub

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/santhoshkumarp1510/AI-Voice-Assistant.git
cd AI-Voice-Assistant
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file:

```
GROQ_API_KEY=your_api_key_here
```

4. Run the assistant:

```bash
python assistant.py
```

---

## 🔐 Environment Variables

Create a `.env` file and add:

```
GROQ_API_KEY=your_api_key_here
```

⚠️ Never share your API key publicly.
